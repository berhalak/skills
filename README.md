# Skills

A collection of Claude Code skills.

## Installation

```bash
npx skills install berhalak/skills
```

This will install all skills from this repository into your Claude Code environment.

To install a specific skill:

```bash
npx skills install berhalak/skills/grainjs
npx skills install berhalak/skills/grist-ui-design
```

## Available Skills

- **grainjs** — Reference for building reactive UIs with the grainjs library. Covers DOM construction, Observables, Computed values, styled components, conditional rendering, event handling, and disposal patterns.
- **grist-ui-design** — Design reference for building UI components in the Grist codebase. Includes design tokens (colors, typography, spacing), icon usage, and component patterns like cards, lists, badges, accordions, and hover effects.
